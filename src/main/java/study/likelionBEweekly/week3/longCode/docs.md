# 기능
1. 사용자에게 인풋 안내메세지
2. input 받기
3. input따라 url 매핑
4. url 따라 기능 실행

### 회원가입
1. id, 비밀번호 input 받기
2. 가입확인
3. db추가
4. 완료 메세지

### 로그인
1. id, 비밀번호 input 받기
2. 가입확인
3. id로 비밀번호 가져오기
4. 입력한 비밀번호와 저장된 비밀번호 같은지 확인
4. 완료 메세지
